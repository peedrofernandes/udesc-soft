Requisitos:
  - Os livros devem ter os seguintes campos: id, titulo, autor, genero, descricao, dos quais o campo descrição é opcional;
  - O usuário deve poder ver todos os livros de maneira uniforme;
  - O usuário deve poder inserir novos livros;
  - O usuário deve poder remover livros existentes;
  - O usuário deve poder editar campos dos livros;
  - Os dados devem ser carregados e salvos localmente pelo usuário;
  - Um conjunto padrão de dados deve ser fornecido;
